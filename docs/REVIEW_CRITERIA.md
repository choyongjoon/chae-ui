# 채 UI — 소개 페이지 디자인 리뷰 기준

리뷰어는 렌더링된 스크린샷만 보고 평가합니다. 코드는 보지 않습니다.

## 1. 시각적 아름다움 (Visual Beauty)

### 색채 조화
- 오방색(청·적·황·백·흑)이 서로 충돌 없이 조화로운가
- 단청 색띠가 실제 전통 단청처럼 격조 있게 보이는가
- 배경(한지색)과 전경 요소 사이 충분한 대비가 있는가
- 색이 너무 많아 산만하거나, 너무 적어 밋밋하지 않은가

### 타이포그래피
- 나눔명조(제목)와 Pretendard(본문)의 조합이 자연스러운가
- 제목과 본문의 크기·굵기 위계가 명확한가
- 행간(line-height)과 자간이 읽기 편한가
- 한글 텍스트가 자연스럽게 렌더링되는가

### 여백과 밀도
- 섹션 사이 여백이 충분한가 (숨 쉴 공간)
- 카드 내부 패딩이 답답하지 않은가
- 전체 페이지 폭이 적절한가
- 요소들이 빽빽하게 몰려있는 곳이 없는가

## 2. 가독성 (Readability)

- 작은 텍스트(12-13px)가 배경 위에서 잘 읽히는가
- 뱃지, 버튼의 텍스트가 배경색 위에서 충분한 대비를 가지는가
- 비활성/muted 텍스트가 너무 희미하지 않은가
- 구분선이 섹션을 명확히 나누는가

## 3. 한국 전통 건축의 아름다움 (Korean Architectural Aesthetics)

### 단청의 원칙
- 색띠가 그라디언트 없이 명확히 구분되는가
- 적·청·황의 배치가 실제 단청의 느낌을 주는가 (적이 가장 바깥, 청이 안쪽)
- 색띠의 비율이 자연스러운가 (균등 분할이 아닌, 주색과 보조색의 구분)

### 창호의 구조감
- 창호 variant 카드가 목재 살대의 느낌을 주는가
- 테두리 두께가 적절한가 (너무 얇으면 창호 느낌이 안 남)
- Dialog의 구조가 창호 문을 연상시키는가

### 한지의 질감
- 배경이 순수한 흰색이 아닌, 한지 특유의 따뜻한 tone인가
- 카드나 오버레이에서 한지의 반투명한 느낌이 드는가

### 전체적 분위기
- "한국 전통 건축"이라는 테마가 페이지를 처음 본 사람에게 전달되는가
- 전통적이면서도 현대적인 균형이 잡혀 있는가 (고리타분하지 않은가)
- 일본/중국 전통 미학과 구별되는 한국적 특징이 있는가

## 4. 창의성 (Creativity)

- 기존 컴포넌트 라이브러리(shadcn/ui, MUI 등)와 시각적으로 차별화되는가
- 전통 소재→UI 변환이 신선하게 느껴지는가
- 놀랍거나 기억에 남는 디테일이 있는가
- 단순히 "한국 색을 입힌" 수준이 아니라, 구조적으로 전통 건축을 반영하는가

## 리뷰 출력 형식

```json
{
  "overall_score": 7,
  "scores": {
    "visual_beauty": { "score": 7, "details": "..." },
    "readability": { "score": 8, "details": "..." },
    "korean_aesthetics": { "score": 6, "details": "..." },
    "creativity": { "score": 7, "details": "..." }
  },
  "top_issues": [
    {
      "priority": "high",
      "area": "visual_beauty",
      "problem": "구체적 문제 설명",
      "suggestion": "구체적 개선 방향"
    }
  ],
  "strengths": ["잘 된 점 1", "잘 된 점 2"],
  "next_actions": ["가장 임팩트 큰 개선 1개"]
}
```

점수 기준:
- 9-10: 출시 가능한 수준, 사람들이 감탄할 만한 디자인
- 7-8: 좋지만 몇 가지 개선하면 훨씬 나아질 것
- 5-6: 기본은 되지만 눈에 띄는 문제가 있음
- 1-4: 근본적 재작업 필요
