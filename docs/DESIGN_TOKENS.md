# 채 UI — 디자인 토큰

## 이름의 의미

**채(Chae)**는 두 한자를 동시에 품습니다.

- **彩** (채색 채) — 고운 빛깔, 채색. 단청에서 건축에 색을 입히는 행위 그 자체. 한국민족문화대백과: "'채'는 색깔 또는 빛의 종류를 말하되 아름답다거나 곱다는 의미를 포함하여 하는 말이다."
- **채** (고유어, 건축 단위) — 한옥에서 건물 한 동(棟)을 세는 고유어. 안채(안쪽 건물), 사랑채(바깥 건물), 별채, 행랑채 등.

## 오방색 (Five Cardinal Colors)

음양오행에 기반한 단청의 기본 5색입니다. UI의 시맨틱 컬러로 매핑됩니다.

| 토큰 | 색명 | Hex | 오행 | 방위 | UI 매핑 |
|------|------|-----|------|------|---------|
| `--cheong` | 청 | #2B6B5E | 木(나무) | 동 | `--primary` |
| `--jeok` | 적 | #C23B3B | 火(불) | 남 | `--destructive` |
| `--hwang` | 황 | #D4A843 | 土(땅) | 중앙 | `--accent` |
| `--baek` | 백 | #F5F0E8 | 金(쇠) | 서 | `--primary-foreground` |
| `--heuk` | 흑 | #2C2825 | 水(물) | 북 | `--foreground` |

## 단청 색띠 프리셋

단청의 색은 **그라디언트가 아닌 명확한 색띠**로 구분됩니다.

- **기본 (default)**: 적·황·청·백 4색 띠
- **금단청 (geum)**: 적·황·청·흑·백 5색 띠
- **가칠 (gachil)**: 청(뇌록) 단색
- **긋기 (gutgi)**: 청 + 흑(먹선) + 백(분선) + 청

## 단청 위계와 Alert

단청의 등급(가칠→긋기→모로→금단청)처럼, Alert 컴포넌트도 색의 위계를 따릅니다.

- **info** = 청(cheong) — 안내, 일반 정보
- **warning** = 황(hwang) — 주의
- **error** = 적(jeok) — 오류, 위험

## 창호 커스터마이징

창호 테두리 두께와 색상은 CSS 변수로 제어합니다:

```css
:root {
  --changho-border: 3px;
  --changho-color: var(--mokjae);
}
```
